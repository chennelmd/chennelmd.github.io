# Darden Healthcare Technology Liaison Website

Static one-page marketing website for Darden Healthcare Technology Liaison.

## Files

- `index.html` - semantic page structure and content
- `styles.css` - responsive styling, brand variables, and layout
- `script.js` - mobile navigation and dynamic copyright year

## GitHub Pages

This site can be published directly from the repository root using GitHub Pages. Replace the placeholder Google Form URL, phone number, privacy policy link, and social links before launch.
